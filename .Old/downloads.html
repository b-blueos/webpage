What we hope to bring with our first release:

  • Blue Package Manager (Name suggestions: BPM, Blue. Language: As of now: shell/bash)
  • A UI which is a little customized to fit the name, BlueOS. (If this is not in the first release, it'll be released later via our package manager.)
  • A working automatic install script. Minimal, but will support:
    • UEFI and BIOS.
    • Architecture: x86_64/amd64 (Only!)
    • Multiboot and clean install.
    • Choosing which drive is used for installation.
  • [Estimated] System requirements:
    • An x86_64 (amd64) processor.
    • 4 GB of RAM
    • 32 GB disk space available. (Although 64 GB is recommended.)
    • Time...
    • A USB-drive to use as installation media.
