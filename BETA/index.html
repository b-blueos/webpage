BlueOS



What is BlueOS?

BlueOS is an Operative System based on Arch Linux. It is a spare-time hobby-project made by some developer with too much free time. We hope to have our first release of the OS in the automn of 2019.
We will release almost every part of the project under the GPL V.3 License, as this project strive for an internet society which can offer software and Operative Systems for free. If you want to collaborate please check out the Contribute-tab.

Should you use BlueOS?

Well, it depends. If you want 100% complete control of what gets installed on your system, then BlueOS is nothing for you. BlueOS is for people who wants a simple distrubution, it is based on Arch, which has the rumor of being har to install, but not to worry! As BlueOS has an automatic script to install the OS without hassle.
