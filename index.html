BlueOS

Welcome to BlueOS!

Introduction

BlueOS is a linux distribution based on Arch LinuxArch Linux is a Linux distro using a rolling release schedule. (Click for more info.).

It is a spare time, hobby, project I am working on, and then mostly for the learning experience!

Right now I, BurnyLlama, is the only one working on this project, but another developer known as Gochady has said he's interested in the project and will probably start working on it too. And together we hope to bring the first release of BlueOS sometime during Q4The last quarter of the year, October, November and December. this year (2019).

This project will be released under the GPL V3 License, as the nature of linux is free and open source software, which we will strive to keep up!

Should you use BlueOS?

Probably not, not yet at least, as BlueOS is not yet done enough. But when it is done you could use it. As of now it is hard to state if there will be any special features in BlueOS... Although we will strive to make BlueOS as customizable as possible, but still leave it as unbloated as possible. As we shall believe in choice! It will include install scripts for Arch Linux which would make an easier/"less worky" way.

From where did the idea originate?

It originated during the spring of 2019, while BurnyLlama was bored and started working a bit on this project. After that the idea was kind of put on the shelf during the summer.

What platforms will BlueOS support?

BlueOS will mostly support UEFI systems, but I will try to make it able to support BIOS too. The UEFI installer will support multibooting, while the BIOS installer will not. Estimated Specifications:

  • 32GB free unallocated space, while 64GB will be recommended.
  • A USB-Drive or other media for the install media.
  • An internet connection.
  • And lastly, time...
