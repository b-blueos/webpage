BlueOS




Introduction:

What is BlueOS?

BlueOS is my very, very, very bad customization of Arch Linux. It is mostly a learning project...
(Using the archiso package from the official repos.)

Who makes BlueOS?

Check out the Crew. You can find it on GitHub.

Who should use BlueOS?

Well... Simply put... Noone would want to use it, unless they would just like a simple way of installing Arch Linux...
(Or if they, for some reason, actually likes some features that I may add... But then again... They would all be available on GitHub and would work with any Arch installation...)

Can BlueOS be installed as an overlay straight onto Arch?

Not yet, but I have ideas to do it. Technically BlueOS is just Arch with some overlays, so I will only have to make a script for it...

Pre-requisites:
  • 32 GB storage space. (64 GB is recommended.)
  • A USB-Drive or CD to be used as installation media...
  • Time...
  • An internet-connection.
  • A UEFI-System, I am currently not supporting regular BIOS... :(
